# 🪑 Móveis Customizados — Landing Page

Landing page de uma loja de móveis sob medida, desenvolvida com HTML e CSS puros. Projeto focado em praticar estruturação de página, estilização e uso de imagens em layouts reais.

---

## 🚀 Sobre o Projeto

O **Móveis Customizados** é uma página institucional fictícia de uma loja de móveis personalizados. O objetivo foi aplicar conceitos fundamentais de HTML semântico e CSS para construir uma página visualmente organizada e funcional.

Desenvolvido como parte dos estudos de introdução ao desenvolvimento web.
